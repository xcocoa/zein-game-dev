# 世界碎片 · World Fragments
> Demo v0.1

---

## 游戏简介

每个人都活在自己的世界里。但有时候，你可以悄悄走进去看看。

玩家扮演一个「世界访客」——进入陌生人留下的生活碎片，
通过文字和图片感受那个人某一刻的日常，并选择要不要留下些什么。

---

## Demo 内容

- ✅ 世界选择界面
- ✅ 世界一：林晓的周五下午（可完整体验）
  - 3 个探索路线（笔记本 / 窗外 / 相机）
  - 留言选择系统
  - 多种结局
- 🔒 世界二、三（锁定，正式版解锁）

## 运行方法

### 方法一：本地运行（推荐）

1. 下载 [Ren'Py SDK](https://www.renpy.org/latest.html)
2. 打开 Ren'Py Launcher
3. 点击 **「添加项目」** → 选择本目录（`myGame/`）
4. 点击 **「启动项目」**

### 方法二：Web 版（浏览器直接玩）

```bash
# 用 Ren'Py SDK 导出 Web 版
./renpy.sh /path/to/myGame web
```

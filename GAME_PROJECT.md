# 游戏项目：文字冒险 iOS 游戏

## 项目定位
- **类型**：文字冒险/视觉小说
- **平台**：iOS（优先）
- **风格**：简约风
- **核心**：剧情驱动 + 高自由度选择 + 属性系统
- **特色**：事件选择影响后续属性，特殊剧情配 CG/视频

## 技术栈选择
- **引擎**：Ren'Py（支持导出 iOS，视觉小说首选）
  - 或 Web 方案：Phaser + React + Cordova 打包 iOS
- **版本控制**：Git（远程仓库同步）
- **美术**：AI 生成 + 简约几何风格

## 开发里程碑
1. [ ] 项目初始化 + Git 远程仓库
2. [ ] 核心框架：属性系统、选择分支
3. [ ] 主线剧情框架
4. [ ] CG/视频集成
5. [ ] iOS 打包测试
6. [ ] 内容填充迭代

## 注意事项
- CG/视频资源文件较大 → 使用 Git LFS
- iOS 打包需要 Mac + Xcode
- 成人内容需注意 App Store 审核政策（可能需要其他分发渠道）
